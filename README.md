# DQ Sentinel

LLM-native data-quality monitoring for enterprise tables and views. Connect a database,
let the system profile your data and propose checks, run them on a schedule, triage
exceptions in a Metabase-style UI, and ask an agentic root-cause analyst to investigate
failures with read-only SQL.

> Full documentation in [AGENTS.md](AGENTS.md) (also the AI-agent harness for this repo).

Build in progress — see the issue tracker and project board.
